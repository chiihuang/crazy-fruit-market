# Installation

## For non-programmer

`publish/html5` is a ready-to-play game file. Use browser (i.e. Chrome, IE, etc) to open `publish/html5/index.html` and here you go!

## For programmer

### Dependancies

- cocos2d-js v3.6.1

Once you setup cocos2d-js environment, you can also do the following:

### Testing

`cocos run -p web`

### Compile

`cocos compile -p web`
